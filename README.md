# Simulador Laboratório Geral

Aplicação Streamlit unificada com duas áreas:

- **Laboratório Central:** capacidade, prazos e pricing de propostas.
- **CTO:** agendamento de férias, suplentes e alocação da equipe.

## Publicação no Streamlit

1. Envie todo o conteúdo desta pasta para um repositório.
2. No Streamlit Community Cloud, selecione `app.py` como arquivo principal.
3. Copie o conteúdo de `secrets_TEMPLATE.toml` para os Secrets do aplicativo.
4. Substitua todos os valores de exemplo pelos valores reais.

Não envie um arquivo real de secrets para o repositório.
